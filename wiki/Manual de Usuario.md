# 📖 Manual de Usuario — WIM [COMM-TIER]

Guía completa para el uso eficiente del sistema de mensajería instantánea del Séquito.

## 1. Comandos de Consola
- `/wim`: Abre el panel de opciones generales.
- `/wim history`: Abre el visor de historial de mensajes.
- `/wim reset`: Restablece todas las opciones por defecto.
- `/wim clean history`: Borra permanentemente el historial de todos los usuarios.

## 2. Gestión de Ventanas
Cada vez que recibas un susurro, se abrirá una ventana dedicada. 
- **Minimizar**: Cierra la ventana (el mensaje se guarda en el icono de minimapa).
- **Cierre Total**: Shift + Click en la 'X' para cerrar la conversación y limpiar el buffer.
- **Enfoque**: Haz click en el texto de entrada para escribir. Pulsa `Tab` para ciclar entre ventanas abiertas.

## 3. Barra de Atajos
En el lateral derecho de cada ventana, encontrarás botones de acceso rápido para:
- Invitar a grupo.
- Inspeccionar equipo.
- Ignorar usuario.
- Consultar información institucional.

## 4. Icono del Minimapa
El icono de la corona muestra el número de mensajes nuevos. 
- **Click Izquierdo**: Abre el menú de conversaciones activas.
- **Click Derecho**: Abre las opciones de WIM.
